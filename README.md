# Quiz 1 - Ingenieria Web - Fronted
Elaborado por Diego Collazos

Examen B

# Uso
Esta aplicación permite:
- Ingresar un nombre (campo de texto).
- Contar la cantidad de veces que se presiona el botón Click.
- Mostrar el total de clicks.
- Guardar un historial de los últimos 5 clicks.
- Reiniciar el contador con el botón Reiniciar.
- Mostrar un mensaje cuando el contador se reinicia.
